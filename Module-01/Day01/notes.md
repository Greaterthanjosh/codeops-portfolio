## GUI and CLI

- CLI (Command Line Interface)
  - Examples: CMD, PowerShell

## Terminal (CLI) Commands

- `pwd` - Shows where you are currently working.
- `ls` - Explores and lists resources/files in the current location.
- `mkdir` - Creates a new folder or directory.
- `touch` - Creates an empty file.
- `echo` - Prints text on the screen or writes text to a file.
- `cat` - Reads and displays the contents of a file.
- `mv` - Moves or renames a file.
- `rm` - Deletes a file.

## Terminal Shortcuts

- `Tab` - Autocompletes commands or file names.
- `Up/Down arrows` - Repeats previous commands.
- `clear` - Clears the terminal screen.
- `history` - Shows previously used commands.
- `Ctrl + C` - Stops a running command.

## Git and GitHub

- Git - Local repository software used to track changes in projects.
- GitHub - An online platform for storing and sharing Git repositories.

## Basic Git Workflow

1. Create a repository on GitHub.
2. Clone the repository to your local machine:

   ```bash
   git clone <url>
   ```

3. Create your own file
   -> touch <your file>
4. Write your name on the file
5. Git add
6. Git commit \_ me <"your msg">
7. Git push
